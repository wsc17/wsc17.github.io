IDEA快捷键

本文介绍：一些默认的快捷键、快捷键的修改、代码提示。

在idea中快捷键设置中，找到（例，introduce Variable），即可修改生成返回值的快捷键设置。

  1. 生成方法的返回值

    introduce Variable

  2. 复制行

    Duplicate Entire Lines

  3. 移动行

    Move Statement Up/Down

  4. 跳转到某行

    Go to Line/Column

  5. 全局搜索快捷键

    search everywhere ---------------> Ctrl + Shift + S

  6. 多行合并为一行

    Ctrl + Shift + j

  7. 回到顶部、底部

    Ctrl + Home
Ctrl + End

  8. 增强for循环代码提示

    iter

  9. 删除无用导入

    Ctrl + Alt + O（这个是英文中的欧，不是数字零）

  10. 删除一行

    Delete Line
